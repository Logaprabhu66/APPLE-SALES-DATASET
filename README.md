Apple Sales Data Analysis Dashboard
1. Introduction
Data analysis helps organizations understand sales patterns, customer demand, and product performance. This project focuses on analyzing apple sales data using data visualization techniques. The dashboard was developed using Microsoft Power BI to convert raw sales data into meaningful insights.
The goal of this project is to analyze apple sales across different countries, apple varieties, and quality grades to identify trends and support better decision-making.

2. Objectives of the Project

The main objectives of this project are:

To analyze apple sales using a dataset.

To visualize sales performance through an interactive dashboard.

To identify top-performing apple varieties and quality grades.

To compare apple sales across different countries.

To provide insights that help improve sales strategies.

3. Dataset Description

The dataset used in this project contains information related to apple sales. The important columns in the dataset include:

Column Name	Description
Record_ID	Unique identifier for each record
Apple_Variety	Type of apple (Fuji, Gala, Golden Delicious, Granny Smith)
Country	Country where apples were sold
Quality_Grade	Category of apple quality (Economy, Premium, Standard)
Quantity_Sold_KG	Quantity of apples sold in kilograms
Price_per_KG_USD	Price of apples per kilogram
Total_KG_Sold	Total kilograms sold
Revenue_USD	Total revenue generated

The dataset helps analyze the relationship between quantity sold, pricing, and revenue.

7. Key Insights from the Dashboard

From the analysis, several insights were discovered:

Standard quality apples generate the highest revenue.

Premium apples also contribute significantly to overall sales.

Apple sales are distributed across multiple countries, showing strong global demand.

Some apple varieties perform better than others depending on the country.

Total revenue from apple sales reached 1.33 million USD.
